ICS 309 General Purpose Communications Log Purpose:

The 309 Communications Log allows you to record details of message traffic and is used by either an individual on an event or a Net Control Operator (NCO). These logs provide the basic reference from which to extract basic logged communications traffic.

Preparation:

The Comm Log is initiated and maintained by the Net Control Operator (NCO) or the individual operator. Completed logs are submitted to the supervisor who forwards them to the Documentation Unit.

Distribution:

The Documentation Unit maintains a file of all Comm Logs. All completed original forms MUST be forwarded to the Documentation Unit.

Comment:

This form is NOT the ICS309 Communications Message Log that is built into Winlink Express. That tool creates a PDF record of inbound/outbound message traffic information.

NOTE on printing.

Printing this form is dependent upon your selected printer and its settings. Various Web Browsers and and their viewing sizes are not all the same. Printing from a browsers print function may be different from the operating systems default print mode. Select what works for you, for both printing and viewing.
